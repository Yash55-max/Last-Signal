1. Product Overview

Product Name: Last Signal (Web Edition)
Platform: Web App + Progressive Web App
Access: Browser-based, installable as PWA
Target Users: Students & digital natives

2. Core Concept (Web Adaptation)

Since web cannot detect battery reliably across all browsers, we adapt:

🔋 Low Battery Mode (If Browser Battery API Available)

Uses Navigator Battery API (if supported)

Activates when battery ≤ 7%

Fallback:

Manual “Low Signal Mode” button

🌙 Midnight Mode

Automatically activates between 12:00–2:00 AM (user local time)

Based on browser time

3. Core Features (MVP)

Anonymous login (Firebase Auth)

Sentiment-based matching

Real-time 1:1 chat

Timer-based session expiration

No message persistence

Installable as PWA

Mobile-friendly UI

4. Success Metrics

Session completion rate

Daily midnight sessions
